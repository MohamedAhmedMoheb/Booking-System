# Ticket Booking System – Project Description

This project is a simple **C++ console application** developed as part of a university program. It simulates a ticket booking system where passengers can book, cancel, and manage tickets based on priority categories.

The system allows users to:

* Book tickets by entering a name and category (VIP, Regular, Economy)
* Cancel existing bookings
* Display all current queues
* Serve passengers based on priority (VIP first, then Regular, then Economy)

The project is designed to demonstrate basic programming concepts such as:

* Object-Oriented Programming (OOP)
* Queue data structures
* Control statements and user interaction

Overall, this project provides a clear example of how priority-based queue systems work in real-life applications like airports or booking platforms.
